Jquery

什么是JQ：一个优秀的JS库，风靡web界很多年，可以说是很多优秀库的参考标准之一。当然，JQ只是辅助工具。

用JQ的好处：

  1. 简化JS的复杂操作
  2. 不用再需要关心兼容性问题
  3. 提供大量的实用方法

如何学习JQ：

  1. 官网：www.jquery.com
  2. 中文API：www.css88.com/jqapi-1.9

JQ设计思想：

  1. 选择网页元素：模拟了CSS选择元素的方法，提供多种筛选方法，且有其独有表达式选择
  2. JQ书写方法：属性函数化，链式操作，取值赋值合体
  3. JQ与JS关系：可以共存，但不能混用

JQ和js写法区别对比（上面是原生js，下面是JQ）：

  1. 赋值：
        document.getElementById("div1").style.background = "red";
    document.getElementsByTagName("div"); //需要遍历
    getByClass(document,"box"); //需要遍历
    $("#div1").css("background","red");
    $("div").css("background","red");
    $(".box").css("background","red");
  2. 页面加载：
        window.onload = function(){};
    $(function(){});
  3. 事件调用：
        document.getElementById("div1").onclick = function(){};
    $("#div1").click(function(){});
  4. this和属性的区别：
        alert(this.innerHTML);
    alert($(this).html());

JQ的选择器（详见api）：

  1. JQ的选择器与css的选择器非常类似，还增加了一些css没有的方法和筛选功能，如
        $('li:eq(2)').css('background','red');
    $('li:odd').css('background','red');
    $('li').filter('.box').css('background','red');
    $('li').filter('[title=hello]').css('background','red');
  2. 注意：上面的$()是JQ的一个全局函数，作为元素选择使用，里面的参数写法如同css；后面的css等都是方法，就是JQ的属性方法化

原生js和JQ的转换：

  1. 错误写法和注意事项：
        alert( $(this).innerHTML );  //错误的写法
    alert( this.html() );       //错误的写法
    oDiv.innerHTML = 'hello';  //js赋值
    alert( oDiv.innerHTML );   //js取值
    $('#div1').html('hello');    //jq赋值
    alert( $('#div1').html() );   //jq取值
    alert( $('li').html() );  //jq获取一组元素的时候，取值是一组中的第一个
    $('li').html('hello');  //jq获取一组元素的时候,赋值是一组中的所有
  2. 原生js和JQ的转换：
        jq对象.get(i)/[i]  //把jq对象转化为原生js对象
    jq对象.get(0)/[0]  //把第一个jq对象转化为原生js对象
    $(原生对象) //把原生js对象转化为jq对象

JQ的链式操作：

  1. 链式操作：
        $('#div1').html('hello').css('background','red').click(function(){
        alert(123);
    });
  2. 里面的方法依次执行，由于每个方法内部设置了return this，因此方法执行后依然得到该对象，可以继续执行其他方法，上面的例子中的结果为对象的内容变成hello（赋值操作），然后背景颜色变成red，最后如果被点击弹出123

$()下的常用方法：

  1. 筛选功能：
        $('div').filter('.box').css('background','red');  //过滤（当前元素）
    $('div').not('.box').css('background','red');   //filter的反义
    $('div').has('.box').css('background','red');   //包含（后代元素）
    $('div').next().css('background','red'); //下一个同级元素
    $('div').find('h2').eq(1).css('background','red'); //后代元素
    alert( $('#h').index() );  //索引是当前元素在所有兄弟节点中的位置
  2. 有些属性没有方法，可以用这个方法取值赋值：
        alert($('div').attr('title'));  //取值
    $('div').attr('title','456');  //赋值
  3. 类名操作：
        $('div').addClass('box2 box4');   //添加类名
    $('div').removeClass('box1');   //移除类名
