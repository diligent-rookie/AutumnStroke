正则表达式的分组/匹配子项：小括号()

  1. 把正则的整体叫母体，然后把左边第一个小括号里的正则，叫做母体的第一个子项，以此类推
  2. 直接操作母体有时会比较麻烦，我们可以把母体分割成多个子项去操作，replace函数的第二个参数就是匹配成功的母体的第一个子项，第三个参数就是匹配成功的母体的第二个子项，以此类推
  3. 下面的例子看看分别会弹出什么：
        var str = '2017-6-7';
    var re = /\d+-/g;（/\d-+/g）（/(\d-)+/g）
    str.replace(re,function($0,$1,$2){
        alert($0);
    })
  4. 匹配重复项\n(n是个大于等于1的数字)，与第n个子项重复的项（复制第n个子项），如：
        var re = /\w\w/;
    var re = /(\w)\1/;
    第一个是两个字符（可以不同），第二个则是相同的两个字符
  5. match用小括号()得到结果：
        var str = 'abc';
    var re = /abc/;（/(a)(b)(c)/）
    alert(str.match(re));
    注意：以上结果是在re不加全局匹配g的时候才能得到

正则表达式匹配相似项：中括号[]

  1. 一组相似的元素，用[]表示，整体代表一个字符，如：
        var str = 'abc';  
    var re = /a[bde]c/;//[]里面是或的关系
    alert(re.test(str));
  2. 连字符(-)：可以是任意从大到小的范围，如[a-z0-9A-Z]，其中任意一个都匹配成功，如：
        var str = 'abc';  
    var re = /a[a-z]c/;
    alert(re.test(str));
  3. 排除(^)：若写在[]里面，则代表排除的意思，如：
        var str = 'abc';  
    var re = /a[^bde]c/;
    alert(re.test(str));
  4. 限定开始位置(^)和限定结束位置($)，他们本身不占位置

正则表达式量词：大括号{}，匹配的字符最少和最多出现的数量

  1. {4,7}--最少出现4次，最多出现7次
  2. {4,}--最少出现4次，最多不限
  3. {4}--恰好出现4次
  4. 特殊情况：
    • +：出现至少1次，完整写法{1,}
    • ?：出现0-1次，完整写法{0,1}
    • *：至少出现0次，完整写法{0,}

正则表达式常用匹配：工作中可以把这些放在一个json里，var jRe = {}，通过jRe.email等方式调用它们

  1. 匹配中文：[\u4e00-\u9fa5]
  2. 行首行尾空格：^\s*|\s*$
  3. Email：^\w+@[a-z0-9]+(\.[a-z]+){1,3}$
  4. 网址：[a-zA-z]+://[^\s]*
  5. QQ号：[1-9][0-9]{4,9}
  6. 邮政编码：[1-9]\d{5}
  7. 身份证：[1-9]\d{14}|[1-9]\d{17}|[1-9]\d{16}x
