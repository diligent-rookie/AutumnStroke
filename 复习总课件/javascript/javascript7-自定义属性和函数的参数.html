JavaScript自定义属性：

  1. 自定义属性：给是给元素上定义本来不存在的属性;
  2. 语法：元素.自定义属性 = 123，可以如原本属性一样进行操作
  3. js可以为任何HTML元素上添加任意数量的自定义属性
  4. 自定义属性是js向html添加东西的方式之一
  5. 例如循环动态给一组按钮添加索引值：元素[i].index = i，索引值的应用--想建立"匹配"/"对应"关系，如数组里的数据和按钮利用索引值一一对应
  6. 重要：后面循环几乎都会用到索引值；因为for循环结束时i已经变成最后一个数字，再次使用i就会出现错误，因此必须用到自定义属性，否则i显示的永远是循环结束后的数字

JavaScript函数的参数：

  1.     function(在此处传参){}  
    fn1(100); function fn1(a){alert(a)};//可以相当于：
    fn1(100); function fn1(var a = 100){alert(100)};//但不能这么写
    其中在函数内部的a叫做形参，而调用函数里的100叫做实参
  2. 也可以传多个参数，如：
        fn1(100,"px"); function fn1(a,b){alert(a+b);}
  3. 参数可以为js的以下数据类型：数字、字符串、布尔值、函数、对象、未定义，网上很多类似的效果就是用函数传参的方式做的
  4. 如下是个对象当参数的例子:
        fn5(window,document); 
    function fn5(w,d){
    	w.onload = function(){
        	d.body.innerHTML=123;
        }
    };
  5. 用参数封装一个方法：
        function fn1(a){
    	if(typeof(a) === 'number' && a===a){
        	alert(a+20);
        }else if(typeof(a)==='string'){
        	alert(a.charAt(2));
        }else if(typeof(a)==='function'){
        	a();
        }
    }
  6. 函数传参应用的注意事项：
    • 尽量保证HTML代码结构一致（否则很难重用代码）
    • 需要把核心程序实现，再用函数包裹起来
    • 把每组中不同的值找出来，通过传参分别实现效果
