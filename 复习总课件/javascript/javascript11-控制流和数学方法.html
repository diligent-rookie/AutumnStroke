JavaScript的控制流：

  1. 流程控制的循环写法：[do ]while/for[ in]（for in会在后面json里详解）
  2. 如下为while的循环：
        var i=0;
    while(i<3){
    	alert(i);
        i++;
    }
    //等同于do while的以下写法（先执行do里的在判断，因此至少会执行一次）：
    var i=0;
    do{
        alert(i);
        i++;
    }
    while(i<3)
    //等同于for的以下写法：
    //注意执行顺序：1声明变量i；2判断；3判断为真时执行代码块；4变量自增
    for(var i=0;i<3;i++){
    	alert(i);
    }
  3. 关键字：break（跳出）、continue（跳过）例子：
        for(var i=0;i<6;i++){
        if(i==4){break;} //当i=4的时候跳出该循环
        alert(i);
    }
    for(var i=0;i<6;i++){
        if(i==4){continue;} //当i=4的时候跳过本次循环，继续后续循环
        alert(i);
    };
  4. 流程控制的判断写法：if/switch
  5. 如下为switch的判断：
        switch(str){
        case "js":break;
        case "html":break;
        default:alert(str);
    }   
    //等同于if的以下写法：
    if(str=="js"){
    }else if(str=="html"){
    }else{
    	alert(str);
    }
  6. 关键字： cass （判断str的值）、 break （跳出判断，既判断成功后不再继续判断后续）、 default（否则，在上面判断都未通过时执行）
  7. with语句：将对象放在with里，代码块中的方法和属性默认是该对象的，如
        with(document){
    	write("hello<br/>");
        write("hi<br/>");
    }//相当于：
    document.write("hello<br/>");
    document.write("hi<br/>");

JavaScript常用的数学方法：

  1. Math.floor()：向下取整（下舍去）
  2. Math.ceil()：向上取整（上进入）
  3. Math.round()：四舍五入
  4. Math.abs()：取绝对值
  5. Math.cos()：余弦值
  6. Math.sin()：正弦值
  7. Math.max()：比较（取大）
  8. Math.min()：比较（取小）
  9. Math.random：0~1的一个随机数
