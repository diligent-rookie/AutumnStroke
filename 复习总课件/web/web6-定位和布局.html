定位

position: relative;相对定位

  1. 不影响元素本身特性（无论区块元素还是内联元素会保留其原本特性）
  2. 不会使元素脱离文档流（元素原本位置会被保留，即改变位置也不会占用新位置）
  3. 没有定位偏移量时对元素无影响（相对于自身原本位置进行偏移）
  4. 提升层级（用z-index样式的值可以改变一个定位元素的层级关系，从而改变元素的覆盖关系，值越大越在上面）

position: absolute;绝对定位

  1. 使元素完全脱离文档流（在文档流中不再占位）
  2. 使内联元素在设置宽高的时候支持宽高（改变内联元素的特性）
  3. 使区块元素在未设置宽度时由内容撑开宽度（改变区块元素的特性）
  4. 相对于最近一个有定位的父元素偏移（若其父元素没有定位则逐层上找，直到document——页面文档对象）
  5. 相对定位一般配合绝对定位使用（将父元素设置相对定位，使其相对于父元素偏移）
  6. 提升层级（同相对定位）

position: fixed;固定定位

  1. 除了相对于可视区偏移外与绝对定位一致
  2. ie6及更低版本不兼容

利用定位做出元素的水平居中、垂直居中和绝对居中

  1. 之前学过的margin:0 auto;——适用范围，子元素比父元素小的时候，只能做水平方向的居中
  2. 利用margin-left和left和定位可以做水平居中，绝对定位或者固定定位会脱离文档流，但是同时可以做垂直居中——该方法适用各种情况，只需要注意是否可以脱离文档流
  3. 用绝对定位或者固定定位配合margin:auto及4个方向偏移值设置为0的方式做绝对居中——适用范围，子元素比父元素小，且需要知道尺寸
布局
  1. 大多数网站会把内容安排到多个列中（就像杂志或报纸那样）
  2. 开始使用布局的时候尽量通过ps等画图工具进行设计
  3. 大多数网站可以使用<div>或者<table>元素来创建多列。CSS 用于对元素进行定位，或者为页面创建背景以及色彩丰富的外观
  4. 虽然我们可以使用HTML table标签来设计出漂亮的布局，但是table标签是不建议作为布局工具使用的 - 表格不是布局工具
  5. 使用 CSS 最大的好处是，如果把 CSS 代码存放到外部样式表中，那么站点会更易于维护。通过编辑单一的文件，就可以改变所有页面的布局
  6. 由于创建高级的布局非常耗时，使用模板是一个快速的选项。通过搜索引擎可以找到很多免费的网站模板（你可以使用这些预先构建好的网站布局，并优化它们）

制作导航

dl、dt、dd定义列表

  1. dt：名字
  2. dd：定义该名字（或详情）

ol有序列表

  1. 不设type属性：数字有序列表
  2. type属性设为A：大写字母有序列表
  3. type属性设为a：小写字母有序列表
  4. type属性设为I：大写罗马数字有序列表
  5. type属性设为i：小写罗马数字有序列表

用清除了默认样式后的ul、ol、li、a等制作导航

  1. ul或ol标签用来承载li标签，并进行嵌套
  2. li标签用来作为导航的每一项
  3. a标签用来做导航的超链接
  4. 如果不希望a标签跳转则将链接里改成"javascript:void(0);"
  5. 通过该变a的hover来改变鼠标移入移出的效果，如可以通过字体颜色、加粗、边框、背景颜色、背景图片等效果来实现
